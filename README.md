# MGC(マルチタスクゲームコミュニティ)


## サイト概要
### サイトテーマ
- 一つのゲームのプレイヤー全体が情報交換や会話する場を
  一つのサイトで複数のゲームをまとめ、閲覧できるようにしたい

### テーマを選んだ理由
- 多種のゲームを同時に遊んでいるとき、
複数のゲーム情報を一つのサイト内で見やすいサイトが
欲しいと思ったから。

### ターゲットユーザ
- 年齢関係なく、ゲームをしている全ユーザ

### 主な利用シーン
- 常時利用したい人が閲覧、チャットをする
